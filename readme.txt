■内容物について

	ColoredFish.exe  : ゲームを起動するためのショートカットです。
	playing.mp4      : ゲームのプレイ動画です。
	Scriptsフォルダ : スクリプトのソースコードが入っています。
	Dataフォルダ    : unityのビルドで作成されるデータが入っています。


■ゲームの概要

	動作環境: Windows
	自機の魚を操作し、自分より小さな魚を食べて一定の大きさを目指します。
	所要時間は1～3分程度。


■起動方法

	「ColoredFish.exe」ショートカットを実行してください。


■操作方法

	方向キー / W, A, S, D : 移動
	Spaceキー             : ゲームスタート、タイトルに戻る
	Rキー                 : リトライ

■ゲームの詳細

	早くクリアするほどスコアが高くなります。
	たまに現れる虹色の魚を食べるとスコアボーナス。
	食べる魚の色を偏らせるとスコアボーナス。
	〇 赤・青・緑と、合成色のシアン・マゼンタ・イエロー 計６パターンが可能です。
	〇 一度偏らせてクリアしたことがある色は記録され、タイトルの文字に着色されていきます。

	データを消すには、[Data]フォルダ内のsave.jsonを削除します。
